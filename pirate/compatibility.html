Compatibility List

The emulator is currently undergoing great changes of its backend which are greatly improving compatibility with games. This compatibility list doesn't reflect the state of the emulator anymore and everyone interested in the emulator is encouraged to re-test their games with the latest emulator builds every few weeks to update the compatibility reports.



Tags: Nothing (0) Bootable (0) Intro (0) Menu (0) Ingame - (0) Ingame + (0) Playable (0) All (0)

Nothing (NaN%): Games that don't boot at all, not displaying an initial black screen with an FPS counter.
Bootable (NaN%): Games that boot but display nothing more than a black screen with an FPS counter.
Intro (NaN%): Games that display any image on screen but doesn't reach to menus.
Menu (NaN%): Games that get into menu but can't enter into gameplay.
Ingame - (NaN%): Games that go into gameplay but have major issues that prevent it from going further early on.
Ingame + (NaN%): Games that go far into gameplay but have glitches that render the game unplayable or have non-playable performance.
Playable (NaN%): Games that can be played from start to finish with playable performance and no game breaking glitches.
